# AIESEC EXPA Helper – IGTa

Operational helper tool to streamline and standardize AIESEC EXPA opportunity creation.

## What this does
This tool helps automate and simplify the creation of EXPA opportunities by:
- Structuring job descriptions
- Standardizing eligibility (skills, backgrounds, study level)
- Reducing repetitive manual work on EXPA

## Intended users
- AIESEC MAHE IGTa team
- Team members with little or no coding background

## Tech stack
- Python
- Streamlit
